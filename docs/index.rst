Justopinion: Download Client for Judicial Opinions
==================================================

Release v\. |release|.

`Justopinion <https://github.com/mscarey/justopinion>`_ is used for loading
and serializing Python objects representing judicial opinions.

Justopinion is available as a Python
package `from the Python Package Index <https://pypi.org/project/justopinion/>`_,
so you can install it from the command line with pip:

.. parsed-literal::
    $ pip install justopinion

Justopinion runs on Python versions 3.8 and up.
